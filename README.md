  <h1>📦 MyListing Frontend Pro (MLF Pro)</h1>

  <p>
    A powerful, lightweight, and user-focused frontend management plugin built specifically for 
    <strong>MyListing WordPress Theme</strong>.  
    MLF Pro enhances listing management, improves usability, and delivers a clean professional UI.
  </p>

  <hr/>

  <h2>✨ Features</h2>

  <h3>🎯 Frontend Listing Management</h3>
  <ul>
    <li>Shortcode-based system (no Elementor dependency)</li>
    <li>Clean and modern UI</li>
    <li>Optimized user experience</li>
  </ul>

  <h3>🔐 Smart Data Handling</h3>
  <ul>
    <li>Hides internal system fields (Track Stats, Job IDs, Package IDs)</li>
    <li>Displays only relevant user data</li>
  </ul>

  <h3>🌐 Social Links Enhancement</h3>
  <ul>
    <li>Fixes “Array Array Array” issue</li>
    <li>Proper social link rendering</li>
    <li>Supports icon-based display (Skyboot icons)</li>
  </ul>

  <h3>🧠 Intelligent Filtering</h3>
  <ul>
    <li>Ignores empty/junk meta fields</li>
    <li>Cleaner frontend output</li>
  </ul>

  <h3>⚙️ Admin Control</h3>
  <ul>
    <li>Admin fields visible in edit mode</li>
    <li>Better control over listing data</li>
  </ul>

  <h3>📧 Email System</h3>
  <ul>
    <li>Structured email handling</li>
    <li>Improved reliability</li>
  </ul>

  <hr/>

  <h2>🏗️ Tech Stack</h2>
  <ul>
    <li>PHP (WordPress Plugin Architecture)</li>
    <li>JavaScript (Frontend interactions)</li>
    <li>CSS (Custom styling)</li>
  </ul>
